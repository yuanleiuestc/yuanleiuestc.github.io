9.1 抽象类抽象方法

包含抽象类方法的类叫抽象类。
当继承一个抽象类的时候时候，必须实现所有的抽象方法。

9.2 接口

· 更进一步的抽象，没有提供任何具体的实现。

  • 使用关键字interface
  • 类实现接口采用关键字implements
  • 接口中的方法都被定义成public,如果没有声明，他们自动也是public

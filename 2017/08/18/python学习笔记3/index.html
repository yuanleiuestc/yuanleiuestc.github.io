#字典

映射：通过名字来引用值的数据结构。

python唯一内建的映射结构是字典，字典中的值没有特殊的顺序，但是都存储在特定的一个键（key）下，键可以是数字，字符串和元组。

字典的创建和使用

创建字典

字典中的键是唯一的，而值并不唯一，通过键来找到对应的值。

字典的创建是由多个键和对应的值构成的键-值对组成，其中每个键和值之间用冒号（：）隔开，项之间用逗号隔开。

dict 函数

通过其它映射（比如其它字典）或者（键，值）对的序列来创建字典
也可以通过关键字创建字典

基本字典操作

字典和和列表的区别

  1. 键的类型，字典的键不一定为整数模型，键的值可以是任意类型，比如:浮点型，字符串或者元组
  2. 自动添加，即使键起初在字典中不存在，也可以直接进行赋值，而列表必须使用append函数，并且还必须在列表的范围之内
  3. 成员资格 ，字典是查找的是键，而不是值。

在字典中检查键的成员资格比在列表中检验值的成员资格更加的高效。

字典的格式化字符串

在% 后面加上键（用括号括起来）

字典的方法

1 clear 清除字典中所有的项

2 copy 返回一个具有相同键值的字典（浅复制）

浅复制当在副本中进行替换时候，原来的字典不受影响，但是，如果修改了某个值，原始的字典也会改变。

深复制用copy模块的deepcopy

3 formkeys 使用给点的键建立字典，每个键都对应一个默认的值None

4 get 根据键值，返回valie的值，如果为空，则返回none
